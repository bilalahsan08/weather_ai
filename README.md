# 🌤️ Weather AI — Flutter Weather App

A beautifully animated Flutter weather app powered by 
Google Gemini AI with dynamic Lottie animations and 
real-time weather data.

## ✨ What Makes This Different

- 🤖 **AI-powered suggestions** using Google Gemini API
- 🎨 **Lottie animations** that change based on weather condition
- 🌈 **Dynamic gradient background** changes with weather and time
- ⏰ **Hourly forecast** with horizontal scroll cards
- 🌙 **Auto day/night detection** with different animations
- 💎 **Glassmorphism UI** — modern frosted glass card design
- ❌ **Clean error handling** for invalid cities

## 📱 Features

- 🔍 Search weather by any city name
- 🌡️ Temperature, humidity, wind speed, feels like, visibility
- 🎬 Lottie animation per condition — sunny, cloudy, rainy, stormy, snowy, foggy, night
- 🤖 AI daily tip generated by Google Gemini based on real weather data
- ⏰ 8-hour forecast with scrollable cards
- 🌈 Background gradient changes automatically with weather
- 🌙 Detects day or night from sunrise/sunset data

## 🚀 Getting Started

### Prerequisites
- Flutter SDK installed
- Android Studio or VS Code
- OpenWeatherMap API key
- Google Gemini API key

### 1. Clone the repository
```bash
git clone https://github.com/bilalahsan08/weather-ai.git
cd weather-ai
```

### 2. Get API Keys

**OpenWeatherMap:**
- Go to [openweathermap.org](https://openweathermap.org)
- Sign up → API Keys → Copy your key

**Google Gemini:**
- Go to [aistudio.google.com](https://aistudio.google.com)
- Click Get API Key → Create API Key → Copy

### 3. Add your API keys

Open `lib/services/weather_service.dart`:
```dart
static const String _apiKey = 'YOUR_OPENWEATHER_KEY';
```

Open `lib/services/ai_service.dart`:
```dart
static const String _apiKey = 'YOUR_GEMINI_KEY';
```

### 4. Download Lottie animations
- Go to [lottiefiles.com](https://lottiefiles.com)
- Download: sunny, cloudy, rainy, stormy, snowy, foggy, night
- Place in `assets/animations/`

### 5. Install dependencies
```bash
flutter pub get
```

### 6. Run the app
```bash
flutter run
```

## 🔑 API Keys Security

Never commit your API keys to GitHub.
Use `--dart-define` to pass keys safely:

```bash
flutter run \
  --dart-define=OPENWEATHER_KEY=your_key \
  --dart-define=GEMINI_KEY=your_key
```
